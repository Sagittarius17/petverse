# **App Name**: PetVerse

## Core Features:

- Pet Adoption Listings: Browse adoptable pets with filtering by species, breed and age.
- Pet Care Guides: Access care guides for various pet types (dogs, cats, birds, etc.).
- Lost & Found Pets: Post and search for lost and found pets.
- User Profile Management: Allow users to register, login and manage their profiles with JWT authentication.
- Image Analyzer: Allow the user to upload an image of their lost pet to help identify possible matches in found pet listings; the AI will act as a tool and identify the key attributes in the image that would aid identification and include those attributes in a summary to the user. The AI should determine whether image analysis is necessary.
- Admin Dashboard: Provide an admin dashboard to manage pets, blogs, categories, users, and adoption posts.
- Database: Employ MongoDB to store and manage application data, including pets, blogs, user data, etc.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke a sense of calm and trustworthiness.
- Background color: Very light blue (#F0F8FF) to keep the design clean and airy.
- Accent color: Light orange (#FFB347) to highlight key actions and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif offering a blend of modernity and warmth, suitable for both headlines and body text.
- Use consistent, friendly icons to represent pet types, actions, and categories.
- Employ a clean, card-based layout with a mobile-first responsive design for optimal viewing on all devices.
- Incorporate subtle animations and transitions to enhance user engagement.
